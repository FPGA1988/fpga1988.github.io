# Jekyll的Gem插件安装方法

## 修改Gemfile
直接输入 `gem 'plugins_name'`

## 安装插件
前提是本地已经安装好了jekyll和ruby的相关环境。cmd下切换到博客根目录，运行`bundle install`即可，运行完之后，会安装指定的插件，然后发现gemfile文件更新，但是发现
插件依旧没有被安装
